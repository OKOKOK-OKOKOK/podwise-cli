## Core Interest Areas

1. **AI / 大模型 / Agent** — 最强兴趣，关注 AI 技术、应用、工具和行业动态
2. **创业 / 投资 / VC** — 关注 startup 生态、融资、投资逻辑和成功 founder 访谈
3. **科技商业 / 行业分析** — 关注科技公司战略、市场竞争格局、行业趋势
4. **宏观经济 / 地缘政治** — 关注全球政经局势、贸易、货币政策等
5. **自我提升 / 思维 / 习惯** — 关注学习方法、思维模型、效率工具

## Listening Style

- **Episode format preference**: 长篇深聊（2h+），偏好马拉松式深度访谈
- **Languages**: 中英混合，两者比例相近
- **Typical listening context**: 专注学习 / 深度工作时段

## What the User Does with Insights

- **Note-taking**: Yes
- **PKM tool**: Obsidian
- **Export habits**: 使用 Obsidian 管理和整理播客笔记和 Highlights

## Output Preferences

- **Preferred format**: 流畅 prose，连贯段落，像写文章
- **Preferred summary length**: medium — 不过分简略也不过于冗长
- **Preferred language for AI output**: Chinese
